## Run Locally

Clone the project

```bash
  git clone https://github.com/RomanMeasv/mp23_astr
```

Go to the project directory

```bash
  cd mp23_astr
```

Install dependencies

```bash
  flutter pub get
```

Start the application

```bash
  flutter run
```
